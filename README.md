# Forge
A platform where developers discover, share, and showcase software projects.



## About

Forge is a modern web platform designed to help developers discover inspiring software projects, showcase their own work, and connect with the developer community.

This project is being built as a portfolio application while following real-world software engineering practices, including reusable components, scalable architecture, Git workflows, and clean code principles.



## Features

### Currently Implemented

- Responsive navigation bar
- Hero section with call-to-action buttons
- Infinite marquee showcasing technologies
- Featured Projects section
- Reusable ProjectCard component
- Dynamic project rendering using external data files
- Pagination for project listings
- Statistics section with reusable StatCard component
- Dynamic Lucide icon rendering
- How It Works section with reusable StepCard component
